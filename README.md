# CultX

Marketing website for **CultX** — the AI-native IP creation and monetization platform.

**Tagline:** Where IPs Become Legends.

## Stack

- Next.js App Router
- GSAP + Lenis scroll motion
- CSS design tokens (purple cinema atmosphere)

## Develop

```bash
npm install
npm run dev    # http://localhost:3000
npm test
npm run lint
npm run build
```

## Routes

| Path | Page |
|------|------|
| `/` | Home |
| `/platform` | Platform loop |
| `/pillars` | Four entertainment formats |
| `/monetize` | Earn paths + token vision |
| `/stars` | Star IPs |
| `/about` | AI Center / Korea |
| `/faq` | FAQ |

## Deploy

Connect this repo to Vercel (Next.js defaults). Root directory is the app root.
